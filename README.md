# Molten-Salt AI Electrolyte Discovery

## Mission
Develop and deploy a custom AI system to discover novel, superior molten-salt electrolytes (chloride, carbonate, or hybrid) for electrochemical graphite synthesis at the lowest possible temperature with maximum carbon solubility, ionic conductivity, stability, and sustainability. Targets outperform LiCl–KCl and Li₂CO₃–Na₂CO₃–K₂CO₃ benchmarks via ML, MD, DFT, and closed-loop experiments.

## Phases
- **Phase 0:** Setup (repo, targets, datasets, ML stack).
- **Phase 1:** Data curation and baseline models.
- **Phase 2:** Generative design.
- **Phase 3:** Simulation validation.
- **Phase 4:** Experimental shortlist.
- **Phase 5:** Closed-loop iteration.

## Key Rules
- Critical honesty: Flag gaps (e.g., data sparsity for hybrids).
- Open-source priority: Use PyTorch, ASE, etc.
- Version control: Commit all major deliverables.
- Safety: Evaluate toxicity/feasibility per prediction.
